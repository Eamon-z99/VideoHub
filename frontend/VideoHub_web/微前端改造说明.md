# VideoHub_web 微前端改造说明

本项目已改造为 qiankun 微前端子应用，可以独立运行，也可以被主应用加载。

## 主要改动

### 1. main.js 改造

- 导出 qiankun 生命周期函数：`bootstrap`、`mount`、`unmount`
- 支持独立运行和作为子应用运行
- 根据运行环境动态设置挂载容器

### 2. 路由配置

- 自动检测 qiankun 环境
- 在 qiankun 环境下使用 `/videohub` 作为 base 路径
- 独立运行时使用默认 base 路径

### 3. Vite 配置

- 添加 CORS 头支持，允许主应用跨域加载
- 配置开发服务器支持跨域访问

## 使用方式

### 独立运行

```bash
npm run dev
```

访问 `http://localhost:5173`

### 作为子应用运行

1. 启动本应用：`npm run dev`
2. 在主应用中访问 `/videohub` 路由

## 注意事项

- 确保路由 base 路径与主应用的 activeRule 匹配
- 生产环境部署时需要配置正确的 base 路径
