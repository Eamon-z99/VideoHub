# 生成测试用户密码指南

## 方法1: 运行测试类（推荐）

在项目根目录执行：
```bash
cd backend/VideoBackend
mvn test -Dtest=PasswordGeneratorTest#generatePasswords
```

## 方法2: 在IDE中运行

1. 打开 `PasswordGenerator.java` 文件
2. 右键点击 `main` 方法
3. 选择 "Run 'PasswordGenerator.main()'"

## 方法3: 使用Maven执行

```bash
cd backend/VideoBackend
mvn compile exec:java -Dexec.mainClass="videobackend.video.util.PasswordGenerator"
```

## 生成的SQL示例

运行工具后，会输出类似以下的SQL语句：

```sql
INSERT INTO `users` (`username`, `account`, `password`, `email`, `status`, `create_time`, `update_time`) 
VALUES ('测试用户2', 'testuser', '$2a$10$xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx', 'testuser@example.com', 1, NOW(), NOW());
```

## 直接使用的SQL（密码: 123456）

如果不想运行工具，可以使用以下SQL（密码是123456，已使用BCrypt加密）：

```sql
-- 测试用户: testuser2 (密码: 123456)
INSERT INTO `users` (`username`, `account`, `password`, `email`, `status`, `create_time`, `update_time`) 
VALUES ('测试用户2', 'testuser2', '$2a$10$N.zmdr9k7uOCQb376NoUnuTJ8iAt6Z5EHsM8lE9lBOsl7iwy8pLNO', 'testuser2@example.com', 1, NOW(), NOW());

-- 测试用户: newuser (密码: 123456)  
INSERT INTO `users` (`username`, `account`, `password`, `email`, `status`, `create_time`, `update_time`) 
VALUES ('新测试用户', 'newuser', '$2a$10$N.zmdr9k7uOCQb376NoUnuTJ8iAt6Z5EHsM8lE9lBOsl7iwy8pLNO', 'newuser@example.com', 1, NOW(), NOW());
```

**注意**: 上面的密码使用了与test用户相同的BCrypt哈希，这是可以的，因为BCrypt可以验证相同的明文密码。
